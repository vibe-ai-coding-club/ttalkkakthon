# TimetableRow

## 목적
타임테이블 단일 행(행사명 + 시간) 표현.

## Props
- `item: { title: string; time: string }`

## 동작/UI
- 모바일에서는 `시간(좌) + 행사명(우)` 가로 배치로 렌더링.
- 데스크탑(`>=768px`)에서는 `행사명(상) + 시간(하)` 세로 배치로 렌더링.
- 모바일에서 행사명은 우측 컬럼 고정폭(`149px`)으로 정렬되고, 시간은 줄바꿈 없이 표시.
- 모바일/데스크탑 각각 타이포 스케일을 다르게 적용.
- 제목은 gray, 시간은 primary 색상으로 강조.
