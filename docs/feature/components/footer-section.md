# FooterSection

## 목적
랜딩 페이지 마지막 브랜드/연락처/소셜 정보 제공.

## 동작/UI
- blue 배경.
- 로고 + 소셜 링크(Instagram/LinkedIn).
- 소셜 링크는 아이콘(`Icon type="instagram" | "linkedin"`)과 텍스트를 가로 배치한다.
- 소셜 링크 내부 간격은 Mobile `4px`, Desktop `6px`을 사용한다.
- 소셜 링크 아이콘 크기는 Mobile `12px`, Desktop `16px`이다.
- 운영 주체 문구 및 문의 이메일 노출.
- 하단 description(운영 주체/문의 이메일)은 모바일에서 항상 세로 스택으로 노출되고, `md` 이상에서 가로 배치된다.
