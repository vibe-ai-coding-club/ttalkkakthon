# SponsorSection

## 목적

후원사 영역과 후원 문의 CTA 제공.

## 동작/UI

- 상단 설명 영역(배경 gray-50) + 후원사 이미지 노출.
- 이 섹션은 `SectionTitle`의 `Chip`을 사용하지 않고 제목 텍스트만 노출.
- 모바일/데스크탑 모두 후원사 이미지가 한 줄에 배치.
- 하단 `Button(color=gray)` 기반 문의 CTA.
  - `mailto:vibecodingclub.team@gmail.com` 링크로 메일 클라이언트 연동.
  - 메일 제목에 `[후원 문의] 띨깍톤 2026` 자동 설정.
