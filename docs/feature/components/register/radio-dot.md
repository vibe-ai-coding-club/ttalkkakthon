# RadioDot

- Source: `app/_components/register/radio-dot.tsx`
- Role: 라디오 인디케이터 원형 도트.
- Props:
  - `checked: boolean`
  - `variant?: "primary" | "gray"` (default: `"primary"`)
- Size: `16x16` (`size-4`), inner dot `8x8` (`size-2`)
- Checked (primary): `border-primary-400`, inner `bg-primary-400`
- Checked (gray): `border-gray-300`, inner `bg-gray-300`
- Unchecked: `border-gray-300`, inner 없음
