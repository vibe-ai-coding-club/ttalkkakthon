# Component Specs

## 운영 규칙

- 이 디렉토리는 컴포넌트별 명세를 개별 파일로 관리합니다.
- 컴포넌트 요구사항/동작/Props/UI가 변경되거나 추가되면 해당 파일을 즉시 갱신합니다.
- 신규 컴포넌트가 생기면 새 명세 파일을 추가하고 아래 인덱스에 등록합니다.
